Delete

Say you have a complex function but you made a simple mistake and want to remove it from existence. You can use the arrow keys select the thing you want to remove and then click ‘del’ to remove it. Let’s remove the O that was entered by mistake in “Hello World!”

Success!

Insert

Now we have removed the O but we need to add the D without overwriting the !. By default your calculator will overwrite any character it is directly above. To get around this select the character directly to the right of where we want our new value to go and then click ‘ins’ and then whatever we want to enter.
