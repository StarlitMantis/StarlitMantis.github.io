Being able to create a table and input x values is very useful. It would be more useful to be able to use functions when we are calculating. Here is how to do so:

a. Click ‘y=’ and then input an equation. In our example we will be using the equation 1/x

b. Go back to the main screen by clicking ‘2nd’ and then ‘Quit’ (under ‘mode’). Once back at the main screen, click ‘alpha’ and then ‘f4’ (under ‘trace’)

c. Select y1 by either pressing ‘enter’ or ‘1’ and then enter a parenthesis ‘(‘. Now you can input any value like 1 or 25, or something more complex like ((25^2)/2). Don’t forget to put your closing parenthesis.
