Saving Variables:

a. You can save any values into a variable. You probably know the most recent answer is saved in ‘ans’ (under (-)) and you can use it like any other number.

b. You have 27 other variables, a-z and theta. You can access these variables by clicking ‘alpha’ and then clicking on any of the letters. You can store values in these variables by having the values on the left and then clicking ‘sto’ and then choosing the variable you prefer.

c. You can use these variables just like ans.

d. Ans is easier to use but you are limited to one variable, by using the other variables made available to you, you can do crazy things. (This ties into writing programs and is a great way to get started conceptually. If you are interested in programming, click here to learn more)

e. Another semi related but important thing to note is that you can use the arrow keys to select past equations you have entered previously, and then click enter on the appropriate one. By doing so you can edit them with delete and insert (talked about in the next section) to save time.
