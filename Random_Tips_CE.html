Random Useful Things to do with the Calculator:

a. There are a couple useful tools on the ‘math’ button screen. Like LogBASE( which allows you to enter any base on a log function.

b. Same deal with xsqrt, or option 5. This allows you to enter any value in the root.

c. When on the ‘math’ screen, click the arrow to the right and you will go to the NUM screen with cool options like abs( which allows you to enter an absolute value)
